# **App Name**: AutoWise

## Core Features:

- Vehicle Health Dashboard: Display a vehicle health score, predicted failure risk, and next service recommendation based on sensor data. Includes confidence level from ML model.
- AI Alert Screen: Display alerts for potential vehicle failures with explainable reasons based on sensor anomalies.
- Auto Service Booking: Suggest a service center and available time slots (AI-selected) for booking services. Allow users to accept or reschedule.
- Voice/Chat AI Assistant: Provide a voice/chat interface for users to ask questions about their vehicle and manage service appointments, using a tool to schedule changes when applicable.
- Predictive Booking Calendar (Service Center): Display a predictive booking calendar with a load heatmap for the next 7-14 days to optimize service center workload.
- Technician Workload Distribution (Service Center): Visualize technician workload distribution to ensure balanced task assignments.
- Recurring Failure Heatmap (OEM): Identify recurring failure patterns and component-wise defect trends using sensor data analysis, including clustering failure through the RCA agent.

## Style Guidelines:

- Primary color: Deep sky blue (#3399FF) to represent reliability and technological innovation.
- Background color: Light blue (#E5F3FF), a very light tint of the primary hue.
- Accent color: Slate blue (#7A89D8), a close, darker tint of the primary blue for interactive elements and key highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clean, modern icons to represent vehicle systems, alerts, and service options.
- Implement subtle animations to provide feedback on interactions and highlight important information.
